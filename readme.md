### windows配置过程

打开v2rayN->服务器->扫描二维码  
扫描我发给你的二维码即可得到配置  
**请私聊我要二维码，二维码会在一段时间内更新，我会把最近更新的日期放在墨墨子那里**


测试方法：选中，Ctrl+O，如果最后有毫秒作为结果，代理就成功了，如果TimeOut请联系我。

![](result.png)

**开始连接**
右键你的右下角V2rayN蓝标->系统代理->自动配置系统代理  
蓝V变红即代表成功  
使用完毕同样位置，清除系统代理  
红V变回蓝色代表关闭代理  
![](blue.png)![](red.png)  
不用记得关，随手好习惯。
